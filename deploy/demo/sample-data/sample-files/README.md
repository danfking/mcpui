# Acme Analytics Platform

A real-time analytics platform for monitoring application performance, user engagement, and business metrics.

## Features

- **Live Dashboards** — real-time metrics with sub-second latency
- **Custom Alerts** — configurable thresholds with Slack/email notifications
- **SQL Query Engine** — ad-hoc analysis on raw event streams
- **REST API** — programmatic access to all metrics and reports

## Quick Start

```bash
npm install
npm run dev
```

See `docs/architecture.md` for system design and `docs/api-reference.md` for endpoints.
